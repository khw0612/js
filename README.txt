week2.js 코드는 split 함수를 사용하여 계산기 코드를 작성하였고 calculator.js 코드는 eval 함수를 사용해서 작성해봤습니다.

week2.js: 
split(" ")는 큰 따옴표 안에 있는 문자 마다 문자열을 분리해주는 역할은 하는 함수이기에  30 + 50를 입력하면 각각 "30" "+" "50"으로 분리됩니다.
이 값들을 expression이라는 변수에 저장하여 계산하는 코드를 작성해 봤습니다.

calculator.js:
eval(str)은 문자열을 넣으면 그대로 문자열을 실행하여 출력하는 함수입니다. 예를 들어 30+50을 입력하게 되면 80을 계산해줍니다.
또한 str.inculdes("+") 함수는 문자열에 해당 문자가 존재하는 지를 판단하는 함수입니다.
예를 들어 expression.includes("+")이라는 코드를 작성했다면 expression이라는 변수(문자열) 안에 +가 있는 지 확인하고 존재하면 true, 존재하지 않으면 false를 나타냅니다.
그래서 이 함수를 활용해서 calculator에 연산 메소드를 작성해 봤습니다.
